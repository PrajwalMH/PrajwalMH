## About

I am **Prajwal M Hulamani**, a **Master’s student in Computer Science at The University of Texas at Arlington** and a software engineer focused on building reliable, scalable, and recruiter-ready engineering systems.

My core strength is **backend software engineering with Java, Spring Boot, REST APIs, MySQL, PostgreSQL, authentication, testing, and distributed system design**. I also build modern full stack products using **Next.js, React, Tailwind CSS, Flask microservices, AWS services, Docker, and CI/CD workflows**.

I enjoy transforming product ideas into production-style systems with clean architecture, secure APIs, strong database design, responsive interfaces, test coverage, and measurable impact. My recent work focuses on **AI-enhanced full stack platforms**, including learning management systems, healthcare triage systems, prescription analysis platforms, recommendation engines, and data-driven dashboards.

I bring a product engineering mindset: understanding user workflows, building end-to-end features, improving performance, writing maintainable code, documenting clearly, and delivering systems that feel polished and enterprise-ready.
